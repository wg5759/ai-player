# 隐私说明

AI播放器采用本地优先设计。当前代码未集成产品分析、广告跟踪或崩溃遥测 SDK。

## 默认留在本机的数据

- 播放历史、项目草稿、拉片点、模型连接配置；
- 用户选择的本地媒体、字幕和导出文件；
- 文档工作台选择的源文件、另存结果和本地操作历史；PDF合并拆分、普通转换和明确公式写入不需要发送给模型；
- 本地 AI 版的模型推理；
- Windows 系统语音合成产生的临时音频。

数据通常保存在 Electron 用户数据目录或用户指定的导出位置。卸载应用不一定删除用户数据目录。

## 何时会发生网络传输

- 用户选择云模型并发起分析、翻译或生成时，提示词以及完成任务所需的字幕、文本或画面描述会发送到所选服务商；
- 文档工作台仅在任务确实需要模型理解或生成内容时发送必要正文，并在当前连接为云模型时要求用户逐次确认；文件路径和未选择文件不作为正文上传；
- 用户显式开启 Wi-Fi 传片、同步或 DLNA 时，应用会在局域网监听服务；
- 用户打开网络媒体地址时，应用会访问该地址及其必要的重定向目标。

不同模型服务商拥有各自的日志、保留和训练政策。用户应在填入 Key 前阅读对应服务条款。项目维护者无法控制第三方服务对数据的处理。

## 密钥

API Key 不应写入项目文件、提交记录、截图、Issue 或日志。导出配置和分享诊断信息前必须人工检查并脱敏。若怀疑密钥泄露，应立即在服务商控制台撤销并轮换。

## 麦克风与屏幕

语音唤醒默认关闭。录音、屏幕捕获和电脑操作只应在用户明确触发后执行；界面关闭不代表操作系统权限已撤销，用户可在系统设置中管理权限。
